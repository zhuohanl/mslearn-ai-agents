
# How to run?

## Step 1: Install the libraries
```
uv sync
```

If want to add `requirements.txt` in a particular lab file:
e.g.
```
uv add Labfiles\02-build-ai-agent\Python\requirements.txt
```

## Step 2: Fill in `.env`

## Step 3: Run
e.g.
```
uv run Labfiles\04-agent-framework\python\agent-framework.py
```
